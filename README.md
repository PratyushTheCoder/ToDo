
# ToDo

A simple flask app which makes managing tasks a less of a burden



## Installation

Clone this repo

```bash
git clone https://github.com/PratyushTheCoder/ToDo.git
```
## Running 

### First activate the virtual-enviorment

```bash
source env/bin/activate
```

### Then simply run the app

```bash
python3 app.py
```
